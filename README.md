# HW 1: SQL and Databases — Team Project 
## Blood Donation Coordination System 

### Spin up environment 

Install Docker Desktop: https://docs.docker.com/desktop/setup/install/mac-install/     
Neighboring pages provide how-to for Windows and Linux, aside from MacOS      

Get DB up and running: `docker-compose up -d`   
Teardown: `docker-compose down`    

### Setting up pgAdmin

Connection:       
Host name: `localhost`   
Username: `admin`   
Password: `admin`   

### Bootstrapping the DB 

1. Run `schema.sql` to create tables    
2. Run `data.sql` to populate DB with mock data
